# Security Policy

## Scope

This public repository contains documentation only.

The product source code, infrastructure code, and deployment internals are hosted in a private repository.

## Reporting A Vulnerability

Please report vulnerabilities privately to: maintainers@example.com

Include:
- A clear description of the issue
- Reproduction steps
- Potential impact
- Suggested mitigation if known

## Sensitive Information

Do not post secrets, tokens, or customer data in public issues.
